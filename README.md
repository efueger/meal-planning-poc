# Meal Planning POC

A simple web application to display my weekly meal plans without needing to log into Google Drive.

## Tech Stack
- **Frontend**: Static HTML/CSS/JavaScript
- **Data Storage**: JSON file
- **Deployment**: GitHub Pages



## Project Structure

```
meal-planning-poc/
├── data/
│   └── current-week.json
├── index.html
├── styles.css
├── script.js
└── README.md
```



## How to Use

### View Locally
1. Clone this repository
2. Open `index.html` in your web browser

### Update Meal Plan
1. Generate new meal plan with Claude
2. Replace contents of `data/current-week.json` with new data
3. Commit and push to GitHub
4. Changes will appear on GitHub Pages automatically
